# php-course-scraper

Scrape UTD Graduate courses

1. Used Python BeautifulSoup for web scraping so added the script to PyScript to run in browser
2. Saved extracted course information in a MySQL table 
3. Built a PHP web page to add and remove courses from the table
4. Implemented dynamic search for course listings
